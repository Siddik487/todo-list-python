# Python To-Do List Application

A simple command-line To-Do List application developed using Python.

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Save tasks permanently using JSON

## Technologies Used

* Python 3
* JSON
* File Handling

## Project Structure

```
todo-list-python/
│
├── main.py
├── tasks.json
├── README.md
└── .gitignore
```

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:

```bash
python main.py
```

## Learning Outcomes

This project demonstrates:

* Functions
* Lists and Dictionaries
* Loops and Conditional Statements
* File Handling
* JSON Data Storage
